What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and design webpages.

Why is HTML important?

It defines the structure of a webpage.
It helps browsers understand and display content.
It can include text, images, links, videos, and more.

Basic HTML Structure

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>
